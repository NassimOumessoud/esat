Embryonic Structural Analysis Tool
==================================

Installing
----------

.. code-block:: shell

    git clone https://github.com/NassimOumessoud/esat.git
    cd esat
    py(thon) setup.py install

Usage
-----

.. code-block:: shell

    py -m esat
